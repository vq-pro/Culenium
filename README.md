# Culenium

Cucumber + Selenium project.

## Features

[Culenium](src/features/Culenium.feature)

## Run Configurations

* Cucumber Scenarios
  * VM arguments: -ea
  * Class: CucumberIT
  * Run Unit Tests first

* Cucumber Scenarios (WIP)
  * VM arguments: -ea -Dcucumber.filter.tags="@WIP"
  * Class: CucumberIT
  * Run Unit Tests first

* Unit Tests
  * All in src/test/java
